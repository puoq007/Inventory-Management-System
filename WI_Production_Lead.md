# Work Instruction (WI)
**รหัสเอกสาร:** WI-PROD-002  
**เรื่อง:** การจัดการชั้นวางและการตรวจสอบสถานะอุปกรณ์ (Locator & Status Management)  
**ผู้ปฏิบัติงาน (Role):** หัวหน้างานฝ่ายผลิต (Production Lead)  
**อุปกรณ์ที่ต้องใช้:** คอมพิวเตอร์ตั้งโต๊ะ, แล็ปท็อป หรือ แท็บเล็ตของบริษัทที่เชื่อมต่ออินเทอร์เน็ต และเครื่องปริ้นเตอร์สำหรับพิมพ์สติกเกอร์ QR Code

---

## 📌 1. การเข้าสู่ระบบ (Login)

1. เปิดแอปพลิเคชัน **Google Chrome** หรือ Web Browser บนคอมพิวเตอร์/แท็บเล็ต
2. พิมพ์ที่อยู่เว็บไซต์: `[ระบุ URL ของระบบตรงนี้]` ในช่องค้นหาด้านบน
3. กรอก **Username** (รหัสพนักงาน) ของท่าน แล้วกดปุ่ม **"Next"**
4. กรอก **Password** (รหัสผ่าน) แล้วกดปุ่ม **"Login"** เพื่อเข้าสู่ระบบ

---

## 📌 2. การจัดการตำแหน่งจัดเก็บ (Locator Management)
*ทำเมื่อต้องการเพิ่มพื้นที่จัดเก็บใหม่ ชั้นวางใหม่ หรือพิมพ์ป้าย QR Code สำหรับชั้นวาง*

1. ที่เมนูด้านซ้ายมือ ให้คลิกเลือกเมนู **"Locator Management"** หรือ **"จัดการชั้นวาง"**
   > *[แทรกรูปภาพที่ 1: หน้าจอด้านซ้ายมือ วงกลมสีแดงรอบเมนู Locator Management]*
2. กดปุ่ม **"Add New Locator"** (เพิ่มตำแหน่งใหม่) ที่มุมบนขวาของหน้าจอ
   > *[แทรกรูปภาพที่ 2: วงกลมสีแดงที่ปุ่ม Add New Locator]*
3. ระบบจะแสดงหน้าต่างให้กรอกข้อมูลพิกัดการจัดเก็บ:
   * **Site:** ระบุโซนหรือพื้นที่หลัก (เช่น Zone A, Line 1)
   * **Cabinet:** ระบุตู้เก็บของ (เช่น Cabinet 01)
   * **Shelf:** ระบุชั้นวางย่อย (เช่น Shelf A, ชั้นที่ 1)
4. ตรวจสอบข้อมูลให้ถูกต้องแล้วกดปุ่ม **"Save"** (บันทึก)
5. เมื่อบันทึกเสร็จสิ้น ตำแหน่งใหม่จะปรากฏในตาราง
6. **การพิมพ์ QR Code ชั้นวาง:**
   * ค้นหาตำแหน่งที่เพิ่งสร้างในตาราง
   * เลือกแถวดังกล่าว แล้วกดปุ่ม **"Print Locator QR"**
   * ระบบจะสร้างหน้า QR Code ให้ตรวจสอบความถูกต้องก่อนสั่งพิมพ์ลงบนสติกเกอร์
   > *[แทรกรูปภาพที่ 3: ตัวอย่างหน้าจอ Preview QR Code ก่อนพิมพ์]*
7. นำสติกเกอร์ QR Code ที่พิมพ์เสร็จแล้ว ไปติดที่ขอบของชั้นวางนั้นๆ ในตำแหน่งที่พนักงาน (Operator) สามารถสแกนได้สะดวกและเห็นชัดเจน

---

## 📌 3. การตรวจสอบสถานะ Jig ในไลน์ผลิต (Status Monitoring)
*ทำเมื่อต้องการตรวจสอบว่า Jig ตัวไหนกำลังถูกใช้งานอยู่ หรือตัวไหนชำรุด*

1. ที่เมนูด้านซ้ายมือ ให้คลิกเลือกเมนู **"Dashboard"** หรือ **"Current Status"**
2. ระบบจะแสดงสถานะภาพรวมของ Jig ทั้งหมด:
   * **Available (พร้อมใช้งาน):** จำนวน Jig ที่อยู่บนชั้นวาง
   * **In Use (กำลังใช้งาน):** จำนวน Jig ที่ถูกเบิกไปอยู่ในไลน์ผลิต
   * **Broken/Needs Cleaning (ชำรุด/รอทำความสะอาด):** จำนวน Jig ที่มีปัญหาและต้องตรวจสอบ
3. หัวหน้างานสามารถค้นหาหรือกรองข้อมูลตามประเภท Jig (Part Type) หรือรหัส Jig (Tool No) เพื่อดูรายละเอียดได้
4. หากพบ Jig ที่สถานะ **Broken** ค้างอยู่เป็นเวลานาน ควรประสานงานกับ **Engineer** เพื่อดำเนินการซ่อมแซมและปลดล็อกในระบบ

---

## ⚠️ ข้อควรระวังและการแก้ปัญหา (Troubleshooting)

* **สติกเกอร์ชั้นวาง (Locator QR) ชำรุดหรือหาย:**
  * ห้ามปล่อยให้ชั้นวางไม่มี QR Code เพราะ Operator จะไม่สามารถสแกนคืน (Scan In) ได้
  * ให้เข้ามาที่เมนู Locator Management ค้นหาตำแหน่งเดิม แล้วสั่ง **Print Locator QR** ใหม่ทันที
* **ลืมรหัสผ่าน:**
  * ให้แจ้ง System Admin เพื่อทำการรีเซ็ตรหัสผ่านใหม่ (ระบบไม่มีปุ่ม Forgot Password หน้าเว็บเพื่อความปลอดภัย)
