# Work Instruction (WI)
**รหัสเอกสาร:** WI-GST-001  
**เรื่อง:** การเข้าใช้งานเพื่อดูข้อมูลและสถานะอุปกรณ์ (Guest Access & Monitoring)  
**ผู้ปฏิบัติงาน (Role):** บุคคลทั่วไป / พนักงานที่ไม่ได้ลงทะเบียน (Guest)  
**อุปกรณ์ที่ต้องใช้:** สมาร์ทโฟน, แท็บเล็ต หรือคอมพิวเตอร์ที่เชื่อมต่ออินเทอร์เน็ต

---

## 📌 1. การเข้าสู่ระบบ (Login)
*ผู้ใช้งานระดับ Guest คือพนักงานที่ยังไม่ได้ถูกเพิ่มชื่อลงในระบบ หรือผู้ที่ต้องการเข้ามาดูข้อมูลเบื้องต้นเท่านั้น โดยไม่ต้องใช้รหัสผ่าน*

1. เปิดแอปพลิเคชัน **Google Chrome** หรือ Web Browser
2. พิมพ์ที่อยู่เว็บไซต์: `[ระบุ URL ของระบบตรงนี้]` ในช่องค้นหาด้านบน
3. กรอก **รหัสพนักงาน (Employee ID)** ของท่านลงในช่อง Username
4. กดปุ่ม **"Next"** หรือกด Enter
5. ระบบจะตรวจสอบ หากไม่พบรหัสพนักงานนี้ในฐานข้อมูล ระบบจะพาท่านเข้าสู่หน้าจอหลักทันทีในฐานะ **"Guest User"** (ไม่ต้องกรอกรหัสผ่าน)

---

## 📌 2. การตรวจสอบข้อมูลบนระบบ (Monitoring)
*Guest สามารถดูข้อมูลภาพรวมได้ แต่จะไม่สามารถทำรายการเบิก-คืน หรือแก้ไขข้อมูลใดๆ ได้*

**1. การดูหน้าแดชบอร์ด (Dashboard):**
* ที่เมนูด้านบน หรือหน้าจอหลัก ให้เลือก **"Dashboard"** (แดชบอร์ด)
* สามารถดูสรุปข้อมูลภาพรวมของ Jig ในระบบทั้งหมด เช่น จำนวน Jig ที่มีอยู่ จำนวนที่ถูกเบิกไปใช้งาน และสถานะคร่าวๆ

**2. การดูสถานะปัจจุบัน (Current Status):**
* ที่เมนูด้านบน ให้เลือก **"Current Status"** (สถานะปัจจุบัน)
* หน้านี้จะแสดงรายละเอียดของ Jig แต่ละตัว ว่าปัจจุบันอยู่ที่ตำแหน่งชั้นวางใด (Locator) หรือถูกเบิกไปใช้งานที่ไลน์ไหน
* สามารถใช้ช่องค้นหา (Search) เพื่อค้นหาตามเบอร์ Jig หรือ Part Type ที่ต้องการได้

---

## ⚠️ ข้อจำกัดสิทธิ์ของ Guest

* **ไม่สามารถเบิก/คืน Jig ได้:** เมนู "Scan In/Out" จะไม่แสดง หากท่านมีความจำเป็นต้องเบิกใช้งาน Jig กรุณาติดต่อ **System Admin** เพื่อขอเพิ่มสิทธิ์เป็น Operator หรือตำแหน่งอื่นๆ ที่เกี่ยวข้อง
* **ไม่สามารถดูประวัติย้อนหลังได้:** เมนู "History" (ประวัติการใช้งาน) จะถูกสงวนไว้สำหรับพนักงานที่มีรายชื่อในระบบเท่านั้น
* **ไม่สามารถเข้าถึงหน้าการจัดการ:** ไม่สามารถลงทะเบียน Jig, พิมพ์ QR Code หรือแก้ไขตำแหน่งชั้นวางได้
