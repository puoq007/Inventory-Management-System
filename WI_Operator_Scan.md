# Work Instruction (WI)
**รหัสเอกสาร:** WI-PROD-001  
**เรื่อง:** การใช้งานระบบแสกนเบิก-คืนอุปกรณ์ช่วยผลิต (Jig Scan Out / Scan In)  
**ผู้ปฏิบัติงาน (Role):** พนักงานฝ่ายผลิต (Operator)  
**อุปกรณ์ที่ต้องใช้:** สมาร์ทโฟน หรือ แท็บเล็ตของบริษัทที่เชื่อมต่ออินเทอร์เน็ต

---

## 📌 1. การเข้าสู่ระบบ (Login)

1. เปิดแอปพลิเคชัน **Google Chrome** บนแท็บเล็ต
2. พิมพ์ที่อยู่เว็บไซต์: `[ระบุ URL ของระบบตรงนี้ เช่น 192.168.1.100]` ในช่องค้นหาด้านบน
   > *[แทรกรูปภาพที่ 1: หน้าจอ Google Chrome พร้อมวงกลมสีแดงที่ช่องกรอก URL]*
3. เมื่อเข้าสู่หน้า Login จะพบกล่องให้กรอก **Username** (รหัสพนักงาน)
   > *[แทรกรูปภาพที่ 2: หน้าจอ Login กล่องแรก พร้อมลูกศรชี้ที่ช่อง Username]*
4. พิมพ์รหัสพนักงานของท่าน แล้วกดปุ่มสีแดง **"Next"**
5. ระบบจะแสดงหน้าให้กรอก **Password** (รหัสผ่าน)
   > *[แทรกรูปภาพที่ 3: หน้าจอ Login กล่องที่สอง พร้อมลูกศรชี้ที่ช่อง Password]*
6. พิมพ์รหัสผ่านของท่าน แล้วกดปุ่มสีแดง **"Login"** เพื่อเข้าสู่ระบบ

---

## 📌 2. ขั้นตอนการเบิก Jig ไปใช้งาน (Scan Out)
*ทำเมื่อต้องการหยิบ Jig ออกจากชั้นวางไปใช้ในไลน์ผลิต*

1. ที่หน้าจอด้านซ้ายมือ (เมนู) ให้ใช้นิ้วจิ้มเลือกเมนู **"Scan Out"** 
   > *[แทรกรูปภาพที่ 4: หน้าจอด้านซ้ายมือ วงกลมสีแดงรอบคำว่า Scan Out]*
2. ระบบจะทำการเปิดกล้อง (หากมีหน้าต่างเด้งถามสิทธิ์การเข้าถึงกล้อง ให้กดปุ่ม **"Allow" หรือ "อนุญาต"**)
3. นำกล้องไปส่องที่ **QR Code บนตัว Jig** ที่ต้องการเบิก ให้ QR Code อยู่ตรงกลางกรอบสี่เหลี่ยมหน้าจอ
   > *[แทรกรูปภาพที่ 5: รูปหน้าจอมือถือตอนเปิดกล้องกำลังสแกน QR บน Jig]*
4. เมื่อสแกนติด ระบบจะแสดงหน้าต่างข้อมูล Jig ตัวนั้นขึ้นมา 
5. ให้ตรวจสอบความถูกต้อง ว่าเบอร์ Jig ตรงกับที่หยิบมาหรือไม่
6. หากถูกต้อง ให้กดปุ่มสีแดง **"Confirm Check-out"** 
   > *[แทรกรูปภาพที่ 6: หน้าต่าง Confirm พร้อมลูกศรชี้ที่ปุ่ม Confirm Check-out]*
7. รอจนกว่าระบบขึ้นข้อความสีเขียวว่า **"Success"** ถือว่าการเบิกเสร็จสมบูรณ์ สามารถนำ Jig ไปใช้งานได้

---

## 📌 3. ขั้นตอนการคืน Jig เข้าชั้นวาง (Scan In)
*ทำเมื่อใช้งานเสร็จแล้ว และต้องการนำ Jig กลับมาเก็บที่ชั้นวาง*

1. ที่หน้าจอด้านซ้ายมือ (เมนู) ให้ใช้นิ้วจิ้มเลือกเมนู **"Home"** (หน้าหลัก)
   > *[แทรกรูปภาพที่ 7: วงกลมสีแดงรอบคำว่า Home ในเมนูด้านซ้าย]*
2. ระบบจะเปิดกล้อง ให้นำกล้องไปส่องที่ **QR Code บนตัว Jig** ที่ต้องการคืน
3. หลังจากสแกน Jig แล้ว หน้าจอจะเปลี่ยนไปเปิดกล้องอีกครั้ง คราวนี้ให้นำกล้องไปส่องที่ **QR Code ประจำชั้นวาง (Locator)** ที่กำลังจะเอาของไปวาง
   > *[แทรกรูปภาพที่ 8: รูปพนักงานกำลังเอากล้องสแกนป้าย QR Code ที่ติดอยู่ตรงขอบชั้นวาง]*
4. ระบบจะแสดงหน้าต่างประเมินสภาพ Jig ก่อนเก็บ จะมี 3 ปุ่มให้เลือก:
   * **ปุ่มสีเขียว (Good):** สภาพปกติ พร้อมใช้งาน
   * **ปุ่มสีเหลือง (Needs Cleaning):** ใช้งานได้ แต่ต้องทำความสะอาดก่อนใช้รอบหน้า
   * **ปุ่มสีแดง (Broken):** Jig เสียหาย แตกหัก หรือชำรุด (ส่งซ่อม)
   > *[แทรกรูปภาพที่ 9: หน้าต่างเลือกสภาพ Jig พร้อมวงกลมครอบทั้ง 3 ปุ่ม]*
5. ใช้นิ้วจิ้มเลือกสภาพตามความเป็นจริง 
6. กดปุ่มสีแดง **"Confirm Check-in"** ด้านล่างสุด
7. รอจนกว่าระบบขึ้นข้อความสีเขียวว่า **"Success"** ถือว่าการคืนเสร็จสมบูรณ์ วาง Jig ลงบนชั้นได้เลย

---

## ⚠️ ข้อควรระวังและการแก้ปัญหา (Troubleshooting)

* **สแกน QR Code ไม่ติด:** 
  * ให้ลองเอามือถือถอยเข้า-ออก เพื่อปรับโฟกัส
  * หากป้าย QR Code ขาดหรือเลือนลาง **ห้ามหยิบไปใช้เด็ดขาด** ให้แจ้งหัวหน้างาน (ProdLead) หรือ Engineer ทันที
* **สแกนผิดตัว ตอนเบิก (Scan Out):**
  * ให้กดปุ่ม **"Cancel"** บนหน้าจอ แล้วทำการสแกนใหม่
* **ลืมรหัสผ่าน:**
  * ให้แจ้งหัวหน้างาน (ProdLead) หรือ Admin เพื่อทำการรีเซ็ตรหัสผ่านใหม่
