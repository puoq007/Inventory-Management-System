# Work Instruction (WI)
**รหัสเอกสาร:** WI-ENG-001  
**เรื่อง:** การลงทะเบียน พิมพ์ QR Code และปลดล็อกสถานะอุปกรณ์ (Jig Registration & Maintenance)  
**ผู้ปฏิบัติงาน (Role):** วิศวกร (Engineer)  
**อุปกรณ์ที่ต้องใช้:** คอมพิวเตอร์ตั้งโต๊ะ หรือ แล็ปท็อปของบริษัทที่เชื่อมต่ออินเทอร์เน็ต และเครื่องปริ้นเตอร์สำหรับพิมพ์สติกเกอร์ทนความร้อน

---

## 📌 1. การเข้าสู่ระบบ (Login)

1. เปิดแอปพลิเคชัน **Google Chrome** หรือ Web Browser
2. พิมพ์ที่อยู่เว็บไซต์: `[ระบุ URL ของระบบตรงนี้]` ในช่องค้นหาด้านบน
3. กรอก **Username** (รหัสพนักงาน) ของท่าน แล้วกดปุ่ม **"Next"**
4. กรอก **Password** (รหัสผ่าน) แล้วกดปุ่ม **"Login"** เพื่อเข้าสู่ระบบ

---

## 📌 2. การลงทะเบียน Jig ใหม่ (Register Jig)
*ทำเมื่อมีการนำ Jig ตัวใหม่เข้ามาในระบบ เพื่อสร้างฐานข้อมูลและออกรหัส*

**แบบที่ 1: การเพิ่มข้อมูลทีละรายการ (Manual Entry)**
1. ที่เมนูด้านซ้ายมือ ไปที่เมนู **"Jig Management"** > เลือก **"Register Jig"**
2. กรอกข้อมูลที่จำเป็นลงในแบบฟอร์ม เช่น:
   * **Tool No:** รหัสของ Jig
   * **Part Type:** ชนิดของชิ้นงานที่ใช้กับ Jig
   * **รายละเอียดอื่นๆ:** ตามที่กำหนด
3. อัปโหลดรูปถ่ายของ Jig (ถ้ามี) โดยกดปุ่มแนบไฟล์
4. ตรวจสอบข้อมูลแล้วกดปุ่ม **"Save"** (บันทึก) ระบบจะสร้างรหัส ID ให้โดยอัตโนมัติ

**แบบที่ 2: การนำเข้าข้อมูลจำนวนมาก (Excel Import)**
1. ที่หน้า **"Register Jig"** คลิกที่ปุ่ม **"Import Excel"**
2. กด **"Download Template"** เพื่อดาวน์โหลดไฟล์ Excel ต้นแบบ
3. กรอกข้อมูล Jig ทั้งหมดลงในไฟล์ Excel ให้ครบถ้วนและถูกต้อง (ห้ามแก้หัวตาราง)
4. อัปโหลดไฟล์ Excel กลับเข้าระบบ และกดปุ่ม **"Confirm"**
5. ระบบจะประมวลผลและนำข้อมูลทั้งหมดเข้าสู่ฐานข้อมูลทันที

---

## 📌 3. การพิมพ์ QR Code สำหรับ Jig (Print QR Code)
*ทำเพื่อสร้างป้าย QR Code สำหรับนำไปติดบนตัว Jig*

1. ค้นหา Jig ที่ต้องการพิมพ์ QR Code ในหน้าตารางแสดงข้อมูล
2. กาเครื่องหมายถูก (✔) เลือก Jig ที่ต้องการ (สามารถเลือกทีละหลายรายการพร้อมกันได้)
3. กดปุ่ม **"Print QR Code"**
4. ระบบจะแสดงหน้าต่าง Preview รูปแบบ QR Code บนกระดาษ (A4)
5. ตรวจสอบความถูกต้อง แล้วสั่งพิมพ์ผ่านเครื่องปริ้นเตอร์ (แนะนำให้พิมพ์ลงบนสติกเกอร์ทนความร้อน)
6. นำสติกเกอร์ไปติดบนตัว Jig ในจุดที่มองเห็นและสแกนได้ง่าย

---

## 📌 4. การจัดการและปลดล็อก Jig ที่ชำรุด (Broken Maintenance)
*ทำเมื่อ Operator สแกนคืน Jig และระบุสถานะว่า "Broken" (ชำรุด) ซึ่งระบบจะทำการล็อก Jig ตัวนั้นไว้ไม่ให้เบิกใช้งานได้อีก*

1. ตรวจสอบหน้า **"Dashboard"** หรือ **"Current Status"** เพื่อค้นหา Jig ที่มีสถานะเป็น **Broken**
2. นำ Jig ตัวดังกล่าวออกจากสายการผลิตและทำการตรวจสอบ/ซ่อมแซม
3. เมื่อซ่อมแซมเสร็จสิ้นและพร้อมใช้งานอีกครั้ง ให้ค้นหา Jig ตัวนั้นในระบบ
4. ทำการอัปเดตสถานะของ Jig โดยเปลี่ยนสถานะจาก **Broken** กลับเป็น **Available** (หรือปลดล็อกตามฟังก์ชันระบบ) เพื่อให้ Operator สามารถสแกนเบิกได้อีกครั้ง

## ⚠️ ข้อควรระวังและการแก้ปัญหา (Troubleshooting)

* **อัปโหลด Excel ไม่ผ่าน:**
  * ให้ตรวจสอบว่าข้อมูลในไฟล์กรอกครบทุกช่องที่เป็นบังคับหรือไม่ และไม่ได้เผลอลบหรือเปลี่ยนชื่อหัวตาราง (Header)
* **QR Code สแกนยากหรือไม่ชัด:**
  * ให้ทำความสะอาดหัวพิมพ์ของเครื่องปริ้นเตอร์ และตรวจสอบคุณภาพของสติกเกอร์ที่ใช้
* **ลืมรหัสผ่าน:**
  * ให้แจ้ง System Admin เพื่อทำการรีเซ็ตรหัสผ่านใหม่
