# Standard Operating Procedure (SOP)
**ระบบจัดการอุปกรณ์ช่วยผลิต (Jig Inventory Management System)**

---

## 1. วัตถุประสงค์ (Purpose)
เพื่อกำหนดมาตรฐานและขั้นตอนการปฏิบัติงานในการใช้งานระบบ Jig Inventory Management System ให้พนักงานทุกระดับสามารถเบิก-คืน ลงทะเบียน และจัดการข้อมูลอุปกรณ์ช่วยผลิต (Jig) ได้อย่างถูกต้อง รวดเร็ว และสามารถติดตามสถานะการใช้งานได้แบบ Real-time ลดปัญหาสินทรัพย์สูญหายหรือหาไม่พบ

## 2. ขอบเขต (Scope)
ระเบียบปฏิบัตินี้ครอบคลุมตั้งแต่การนำเข้าข้อมูล Jig การสร้างชั้นวาง (Locator) การพิมพ์บาร์โค้ด (QR Code) ไปจนถึงขั้นตอนการสแกนเบิก-คืนของพนักงานฝ่ายผลิต (Operator) ในสายการผลิตทั้งหมด

## 3. หน้าที่ความรับผิดชอบ (Roles & Responsibilities)
ระบบได้จำกัดสิทธิ์การเข้าถึงตามตำแหน่งงาน ดังนี้:

*   **System Admin:** ดูแลภาพรวมของระบบ, เพิ่ม/ลด ผู้ใช้งาน (User), รีเซ็ตรหัสผ่าน, และมีสิทธิ์แก้ไขข้อมูลทุกอย่างในระบบ
*   **Engineer:** รับผิดชอบการ **เพิ่ม/แก้ไข ข้อมูลสเปกของ Jig**, อัปโหลดรูปภาพ Jig, นำเข้าข้อมูลผ่าน Excel และพิมพ์ QR Code ของ Jig
*   **Production Lead (ProdLead):** รับผิดชอบการ **จัดการชั้นวาง (Locator)**, กำหนดพิกัด (Site > Cabinet > Shelf), พิมพ์ QR Code ประจำชั้นวาง และตรวจสอบสถานะ Jig ในไลน์ผลิต
*   **Operator:** ผู้ปฏิบัติงานหน้างาน มีหน้าที่ **สแกนเบิก (Check-out)** และ **สแกนคืน (Check-in)** Jig ตามความเป็นจริง ห้ามนำ Jig ออกไปโดยไม่สแกนเด็ดขาด

## 4. ขั้นตอนการปฏิบัติงาน (Operating Procedures)

### 4.1 การเข้าสู่ระบบ (Login)
1. เปิด Web Browser (แนะนำ Google Chrome) บนแท็บเล็ตมือถือ หรือคอมพิวเตอร์
2. พิมพ์ URL ของระบบ (เช่น `http://[IP-Address]`)
3. กรอก **Username** และกด "Next"
4. กรอก **Password** และกด "Login"
5. ระบบจะพาท่านไปยังหน้าจอที่สอดคล้องกับสิทธิ์การใช้งานของท่าน

### 4.2 การลงทะเบียน Jig และพิมพ์ QR Code (สำหรับ Engineer)
**กรณีเพิ่มข้อมูลทีละรายการ (Manual Entry):**
1. ไปที่เมนู **Jig Management** > เลือก **Register Jig**
2. กรอกข้อมูลบังคับ: Tool No, Part Type, และรายละเอียดอื่นๆ
3. แนบรูปถ่ายของ Jig (ถ้ามี)
4. กดปุ่ม **Save** ระบบจะสร้างรหัส ID ให้โดยอัตโนมัติ

**กรณีนำเข้าข้อมูลจำนวนมาก (Excel Import):**
1. ไปที่เมนู **Register Jig** > คลิกปุ่ม **Import Excel**
2. ดาวน์โหลด Template และกรอกข้อมูลให้ครบถ้วน
3. อัปโหลดไฟล์กลับเข้าระบบและกด **Confirm**
4. ระบบจะประมวลผลและสร้างข้อมูล Jig ทั้งหมดเข้าสู่ Database ทันที

**การพิมพ์ QR Code (Jig):**
1. ค้นหา Jig ที่ต้องการในตาราง
2. เลือก Jig (สามารถเลือกหลายตัวพร้อมกันได้)
3. กดปุ่ม **Print QR Code**
4. ตรวจสอบความถูกต้องของหน้ากระดาษ (A4) แล้วสั่งพิมพ์ลงบนสติกเกอร์ทนความร้อน

### 4.3 การจัดการตำแหน่งจัดเก็บ (สำหรับ ProdLead)
1. ไปที่เมนู **Locator Management**
2. กด **Add New Locator** เพื่อสร้างพื้นที่เก็บใหม่
3. กำหนดชื่อ Site, Cabinet (ตู้) และ Shelf (ชั้น)
4. กด **Save** และเลือก **Print Locator QR**
5. นำสติกเกอร์ QR Code ไปติดที่ขอบของชั้นวางนั้นๆ ให้เห็นชัดเจน

### 4.4 การเบิก Jig ไปใช้งาน (Scan Out - สำหรับ Operator)
1. ล็อกอินเข้าสู่ระบบด้วย Username/Password ของตนเอง
2. ไปที่เมนู **Scan Out**
3. ระบบจะเปิดกล้อง (หรือใช้เครื่องสแกนบาร์โค้ด)
4. **สแกน QR Code บนตัว Jig** ที่ต้องการเบิก
5. หน้าจอจะแสดงข้อมูล Jig ให้ตรวจสอบความถูกต้อง
6. กดปุ่ม **Confirm Check-out**
7. สถานะของ Jig จะเปลี่ยนเป็น **"In Use"** (กำลังใช้งาน)

### 4.5 การคืน Jig เข้าชั้นวาง (Scan In - สำหรับ Operator)
1. ไปที่เมนู **Home** หรือหน้าสแกนหลัก
2. **สแกน QR Code บนตัว Jig** ที่ต้องการส่งคืน
3. ระบบจะถามหาพิกัดจัดเก็บ ให้ทำการ **สแกน QR Code ประจำชั้นวาง (Locator)** ที่กำลังจะนำ Jig ไปวาง
4. ประเมินสภาพของ Jig ก่อนเก็บ โดยเลือกจาก 3 ตัวเลือก:
    *   `Good` (สภาพปกติ พร้อมใช้งาน)
    *   `Needs Cleaning` (ต้องทำความสะอาดก่อนใช้งานครั้งต่อไป)
    *   `Broken` (ชำรุด แจ้งซ่อม)
5. กดปุ่ม **Confirm Check-in**
6. สถานะของ Jig จะเปลี่ยนเป็น **"Available"**

## 5. ข้อควรระวังและกรณีฉุกเฉิน (Exceptions & Precautions)

1. **กรณี QR Code ฉีกขาด/สแกนไม่ติด:**
    *   **ห้าม** นำ Jig ไปใช้งานโดยไม่สแกน
    *   ให้ Operator พิมพ์รหัส ID ของ Jig ด้วยมือ (Manual Input) ในหน้าสแกน หรือแจ้ง Engineer/ProdLead ให้พิมพ์ QR Code ดวงใหม่ทันที
2. **กรณีพบ Jig ชำรุด (Broken):**
    *   ตอนสแกนคืน (Scan In) ให้ Operator ระบุสภาพเป็น `Broken`
    *   ระบบจะทำการล็อก Jig ตัวนั้นไม่ให้มีการสแกนเบิกออกไปใช้ได้อีก จนกว่า Engineer จะเข้ามาตรวจสอบและกดปลดล็อก (ซ่อมเสร็จแล้ว)
3. **การเปลี่ยนรหัสผ่าน:**
    *   ผู้ใช้สามารถเปลี่ยนรหัสผ่านของตนเองได้ที่เมนู **Profile**
    *   หากลืมรหัสผ่าน ให้แจ้ง System Admin เพื่อทำการรีเซ็ต (ระบบไม่มีปุ่ม Forgot Password หน้าเว็บเพื่อความปลอดภัย)
