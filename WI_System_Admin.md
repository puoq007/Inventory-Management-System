# Work Instruction (WI)
**รหัสเอกสาร:** WI-ADM-001  
**เรื่อง:** การจัดการผู้ใช้งานและดูแลระบบ (User & System Management)  
**ผู้ปฏิบัติงาน (Role):** ผู้ดูแลระบบ (System Admin)  
**อุปกรณ์ที่ต้องใช้:** คอมพิวเตอร์ตั้งโต๊ะ หรือ แล็ปท็อปของบริษัทที่เชื่อมต่ออินเทอร์เน็ต

---

## 📌 1. การเข้าสู่ระบบ (Login)

1. เปิดแอปพลิเคชัน **Google Chrome** หรือ Web Browser
2. พิมพ์ที่อยู่เว็บไซต์: `[ระบุ URL ของระบบตรงนี้]` ในช่องค้นหาด้านบน
3. กรอก **Username** (รหัสพนักงาน) ของแอดมิน แล้วกดปุ่ม **"Next"**
4. กรอก **Password** (รหัสผ่าน) แล้วกดปุ่ม **"Login"** เพื่อเข้าสู่ระบบ

---

## 📌 2. การจัดการผู้ใช้งานระบบ (User Management)
*ทำเมื่อต้องการเพิ่มพนักงานใหม่เข้าสู่ระบบ, ลบสิทธิ์พนักงานที่ลาออก หรือปรับเปลี่ยนตำแหน่ง*

**การเพิ่มผู้ใช้งานใหม่ (Add User):**
1. ที่เมนูด้านซ้ายมือ ไปที่เมนู **"User Management"** หรือ **"ตั้งค่าระบบ"** > เลือก **"Users"**
2. กดปุ่ม **"Add New User"** (เพิ่มผู้ใช้ใหม่)
3. กรอกข้อมูลส่วนตัวของพนักงาน เช่น:
   * **Username:** รหัสพนักงาน (ใช้สำหรับ Login)
   * **Name/Surname:** ชื่อ-นามสกุล
   * **Role:** เลือกระดับสิทธิ์การใช้งาน (System Admin, Engineer, ProdLead หรือ Operator)
4. กำหนด **Password** เริ่มต้นให้พนักงาน
5. กดปุ่ม **"Save"** (บันทึก) เพื่อสร้างบัญชี

**การลบหรือระงับสิทธิ์ผู้ใช้ (Deactivate User):**
1. ค้นหารหัสพนักงานหรือชื่อของพนักงานในหน้า User Management
2. กดปุ่ม **"Edit"** (แก้ไข) หรือ **"Deactivate"** (ระงับการใช้งาน)
3. เปลี่ยนสถานะบัญชีเป็น Inactive (ปิดการใช้งาน) หรือกด Delete (หากระบบรองรับ)
4. กด **"Save"** (บันทึก)

---

## 📌 3. การรีเซ็ตรหัสผ่าน (Reset Password)
*ทำเมื่อมีผู้ใช้งานลืมรหัสผ่าน เนื่องจากระบบปิดฟังก์ชัน Forgot Password ไว้เพื่อความปลอดภัย*

1. เมื่อได้รับแจ้งจากผู้ใช้งาน ให้ไปที่เมนู **"User Management"**
2. ค้นหาชื่อ หรือ รหัสพนักงาน (Username) ของผู้ที่ต้องการรีเซ็ตรหัสผ่าน
3. กดปุ่ม **"Edit"** (แก้ไข) ที่ชื่อของผู้ใช้งานรายนั้น
4. ทำการกรอกรหัสผ่านใหม่ลงในช่อง **Password** (อาจใช้รหัสผ่านชั่วคราวที่เป็นมาตรฐาน เช่น `123456`)
5. กดปุ่ม **"Save"** (บันทึก)
6. แจ้งรหัสผ่านใหม่ให้ผู้ใช้งานทราบ และแนะนำให้ผู้ใช้งานเปลี่ยนรหัสผ่านทันทีเมื่อล็อกอินเข้าสู่ระบบสำเร็จ (ที่เมนู Profile)

---

## 📌 4. การจัดการข้อมูลหลักและการตั้งค่าระบบ (Master Data Management)
*System Admin มีสิทธิ์สูงสุดในการเข้าถึงและแก้ไขข้อมูลทุกอย่าง หากเกิดข้อผิดพลาดในการลงข้อมูลจาก User ระดับอื่น*

1. **การแก้ไขข้อมูล Jig ที่ผิดพลาด:** สามารถเข้าไปที่เมนู **Jig Management** เพื่อแก้ไขข้อมูลที่ Engineer ลงไว้ผิดพลาดได้
2. **การแก้ไขข้อมูลชั้นวาง (Locator):** สามารถเข้าไปที่ **Locator Management** เพื่อลบหรือแก้ไขพิกัดที่ ProdLead สร้างไว้ได้
3. **การตรวจสอบ Log:** (ถ้ามี) Admin สามารถตรวจสอบประวัติการสแกน หรือประวัติการเข้าใช้งานระบบของผู้ใช้แต่ละรายได้ เพื่อใช้ตรวจสอบกรณี Jig สูญหาย

---

## ⚠️ ข้อควรระวังในการปฏิบัติงาน

* **การกำหนดสิทธิ์ (Role Assignment):**
  * ต้องระมัดระวังในการมอบสิทธิ์ Admin หรือ Engineer ให้กับพนักงาน ต้องตรวจสอบให้แน่ใจว่าได้รับอนุมัติจากผู้จัดการหรือผู้ที่เกี่ยวข้อง
* **ความปลอดภัยของบัญชี Admin:**
  * ห้ามแชร์ Username และ Password ของ Admin ให้ผู้อื่นเด็ดขาด
  * ควรตั้งรหัสผ่านของ Admin ให้คาดเดาได้ยากและเปลี่ยนรหัสผ่านทุกๆ 3-6 เดือน
